## 运行方式
```bash
# clone项目到本地
git clone https://github.com/corty919/My-chat-room.git

# 安装项目依赖
npm install

# 运行以下命令打开浏览器的localhost:3000查看
cd My-chat-room
node server.js
```
## 技术栈
> node+express+socket.io+jquery+flex

## 目前实现功能
- 登录检测
- 系统提示在线人员状态（进入/离开）
- 接收与发送消息 
- 自定义消息字体颜色
- 支持发送表情
- 支持发送图片 
- 支持发送窗口震动
